# Mind Space

Mind Space — минималистичное локальное веб-приложение для дневника эмоций.

## Возможности

- Ввод настроения (с авторскими строгими иконками), интенсивности (1–10), заметки и тегов.
- История записей с редактированием и удалением.
- Аналитика: heatmap за 90 дней, линейный график за 7/30 дней, облако тегов.
- Настройки: экспорт JSON/CSV, импорт JSON/CSV, сброс всех данных.
- Полностью офлайн: данные хранятся в `localStorage`, сервер не требуется.
- Темная космическая тема и адаптивный интерфейс.

## Запуск

Откройте `index.html` в браузере
или запустите локальный сервер:

```bash
python3 -m http.server 4173
```

и перейдите на `http://localhost:4173`.
